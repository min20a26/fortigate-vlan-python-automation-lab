# FortiGate VLAN Security & Python Automation Lab

## Overview
This project demonstrates a home lab built with FortiGate 40F, Cisco ASR1001-X, and a managed switch. The lab focuses on VLAN segmentation, firewall policies, SSH management, logging, and Python automation.

## Lab Equipment
- FortiGate 40F
- Cisco ASR1001-X
- Managed Switch
- Dell Laptop (Windows)
- MacBook Air

## Features
- VLAN 10 and VLAN 20
- FortiGate Firewall Policies
- SSH Access to Cisco ASR
- Python Automation
- QUIC Blocking
- Firewall Logging
- Network Documentation
